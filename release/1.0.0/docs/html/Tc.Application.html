Class Tc.Application

Responsible for application wide issues (ie. the creation of modules).

Constructor

Tc.Application ( )

Properties

$ctx - jQuery

The jquery context.

config - Object

The configuration.

connectors - Object

Contains references to all connectors on the page.

modules - Array

Contains references to all modules on the page. -> could be useful ie. when there are interactions between flash <-> js.

sandbox - Sandbox

The sandbox to get the resources from (shared between all modules).

wildcardComponents - Array

Contains references to all wilcard components on the page.

Methods

init

void init ( $ctx , config )
Initializes the Application.
Parameters:
$ctx <jQuery> the jquery context
config <Object> the configuration
Returns: void

registerConnection

void registerConnection ( connector , component )
Registers a connection between a module and a connector.
Parameters:
connector <String> the full connector name (ie. MasterSlave1Slave).
component <Module> the module instance.
Returns: void

registerModule

Module registerModule ( $node , modName , skins , connectors )
Registers a module.
Parameters:
$node <jQuery> the module node.
modName <String> the module name. it must match the class name of the module (case sensitive).
skins <Array> a list of skin names. each entry must match a class name of a skin (case sensitive)).
connectors <Array> a list of connectors identifiers. schema: (ie. MasterSlave1Master).
Returns: Module
the reference to the registered module.

registerModules

Array registerModules ( $ctx )
Registers all modules in the context scope automatically (as long as the modules uses the oocss naming conventions).
Parameters:
$ctx <jQuery> the jquery context.
Returns: Array
a list containing the references of the registered modules.

start

void start ( modules )
Starts (intializes) the registered modules.
Parameters:
modules <Array> a list of the modules to start.
Returns: void

stop

void stop ( modules )
Stops the registered modules.
Parameters:
modules <Array> a list containting the module instances to stop.
Returns: void

unregisterModule

void unregisterModule ( modules )
Unregisters the modules by the given module instances.
Parameters:
modules <Array> a list containting the module instances to unregister.
Returns: void
